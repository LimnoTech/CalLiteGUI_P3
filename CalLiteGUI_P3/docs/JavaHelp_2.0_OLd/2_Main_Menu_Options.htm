2. Main Menu Options

The majority of user choices in the CalLite GUI are handled on a series of dashboards accessed through tabs across the top of the interface. The six dashboards for specifying scenario settings contain black text (Run Settings, Hydroclimate, Demands, Facilities, Regulations, Operations), while the five dashboards for viewing results contain blue text: (Quick Results, Custom Results, Map View, External PDF, Web Map). See Figure 2-1.

 

There are two dropdown menus:

 

 File: Options to load, run, save, save as, or delete an existing scenario.

 

 Help: Access to the help system and the current CalLite GUI version information.
