Detailed Description

Class to grab (load) DSS time series for a set of scenarios passed in a JList. Each scenario is a string that corresponds to a fully qualified (?) DSS file name. The DSS_Grabber instance provides access to the following:

  • Main time series (result, including necessary math) for each scenario
  • Secondary series (control) where indicated for each scenario
  • Difference for main time series for each scenario
  • Exceedance for main time series for each scenario

Typical usage sequence:

  • DSS_Grabber
  • setIsCFS
  • setBase
  • setLocation
  • setDateRange
  • getPrimarySeries
  • getSecondarySeries
  • Other calculations
